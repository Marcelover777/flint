Show the flint quick-reference card.

| Command | What |
|---|---|
| `/flint` | Activate at default level (full) |
| `/flint lite` | Light compression — ~30% tokens dropped |
| `/flint ultra` | Maximum compression |
| `/flint wenyan[-lite\|-ultra]` | Classical Chinese compression |
| `/flint off` | Deactivate |
| `/flint-commit` | Terse commit message |
| `/flint-review` | One-line review findings |
| `/flint-compress <file>` | Compress a Markdown file |
| `/flint-stats` | Lifetime token-savings |

Natural language also works: "turn on flint", "stop flint", "normal mode".

Code, commits, security warnings: flint drops out automatically.
